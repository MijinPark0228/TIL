## Quick Sort (**クイックソート**)

#### 説明

- 특정 원소를 기준으로 주어진 배열을 두 부분배열로 분할하고, 

  각 부분배열에 대해서 퀵정렬을 순환적으로 적용하는 방식

- 피벗이 제자리를 잡도록하여 정렬하는 방식

  피벗(pivot) : 두 부분배열로 분할할 때 기준이 되는 특정 원소

  

  |      |      |      |      |      |      |      |      |      |      |      |
  | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
  |      |      |      |      |      |      |      |      |      |      |      |
  |      |      |      |      |      |      |      |      |      |      |      |
  
  
  
  왼쪽 부분배열의 모든 값 < 피벗 < 오른쪽 부분배열의 모든 값
  
  

#### 性能 

`T(n)=T(n/2)+Θ(1), T(1)=Θ(1) → Θ(logn)`
